WELCOME TO THE TIMMY PLUGIN PAGE


This page contains a fully simple explanation about a PHPRunner plugin. The intention of making this object is to offer another way of entering datetime data in PHPRunner and also to contribute to make Xlinesoft Marketplace bigger. This plugin is written entirely in PHP and CSS, using JAVASCRIPT JQUERY and this fantastic plugin: bootstrap-material-datetimepicker.js.

It's the eighth plugin I've made. I hope you to enjoy programming using this little tool, it will save you hours in front of your screen. Feel free to send your opinion. You can view more MIB Plugins for PHPRunner soon.




COMPATIBILITY AND VERSIONS


This plugin was made only for PHPRunner.
Tested in version 8.0.
It uses the standard structure for all plugins, so you could use in any other version.




INITIAL PARAMETERS

As any other plugin, you have to initialize it providing certain parameters to let the control to function properly as your needs. Many of them are self-explanatory. Here's the list of parameters for version 1.0:

"format": MomentJS Format. Check http://momentjs.com
"shortTime": Display 12 hours AM|PM
"minDate": Minimum selectable date
"maxDate": Maximum selectable date
"currentDate": Initial date (string)
"date": Has datepicker
"time": Has timepicker
"clearButton": Show clear button
"nowButton": Show now button
"switchOnClick": Switch view on click
"cancelText": Text for the cancel button
"okText": Text for the OK button
"clearText": Text for the clear button
"nowText": Text for the now button
"lang": Language for displaying names
"weekstart": Beggining day of week

When you initialize the plugin you'll see examples commented for all parameters. It's very simple to use !
